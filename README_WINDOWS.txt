BeebEm for Microsoft Windows
============================

Compiling
---------

This distribution of BeebEm contains Windows executables but if you want to
compile BeebEm yourself then you will need Microsoft Visual Studio 2005 or
later (the free VS2005 Express will compile BeebEm).  The following project
files are included:

  BeebEm.sln     - Solution file
  BeebEm.vcproj  - Project file

You will need to download and install the following SDKs:

1. Microsoft Windows Server 2003 SP1 Platform SDK

2. Microsoft DirectX 9.0 SDK

3. Microsoft Speech SDK 5.1
